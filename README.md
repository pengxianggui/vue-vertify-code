# vue-vertify-code
vue前端图形验证码

## 使用方法
```vue
import VertifyCode from 'verification-code'
<template>
    <vertify-code></vertify-code>    
</template>

export default {
    components: {
        VertifyCode
    }
}
```

## 说明
图片来源自互联网: https://picsum.photos ———— 一个可根据url中参数获取图片和长宽的网站
